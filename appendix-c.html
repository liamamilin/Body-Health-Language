Appendix C: Sentence Pattern Library

This appendix provides reusable sentence patterns organized by function. These patterns translate the concepts of body-health language into daily use. For each category, patterns are ordered from basic to more specific. Use these patterns to describe body state, assess conditions, make decisions, and communicate about health.
