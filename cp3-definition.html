4  Definition

Not all expression works by showing what something is like. Sometimes language must state what something is.

Description gives shape, texture, and perceptible form. It tells us how a thing appears, feels, or presents itself. But this is not always enough. Some kinds of thought require a stricter act: the setting of a boundary. A term must be fixed, a concept must be clarified, a category must be marked, and the reader must be told not only what something resembles, but what counts as belonging to it.

This is why definition matters.

A weak expression often fails here by relying only on loose description. It gives examples, attributes, and impressions, but never stabilizes the term itself. The result is that later statements float. Classification becomes uncertain, comparison loses its basis, and explanation becomes harder to build. Definition corrects this by establishing the object more tightly. It says, in effect: this is what we mean here.

4.1 What is a definition?

At the level of ordinary language, a definition is an expression that states what a term, thing, or concept is by identifying its essential features and its boundary.

We notice definition through simple patterns such as:

  • defining a term directly

  • identifying essential features

  • setting boundaries

  • distinguishing definition from looser description

Examples include:

  • A triangle is a shape with three sides.

  • A teacher is a person whose role is to teach.

  • Sleep is a state of rest in which awareness is reduced.

  • A tool is an object used to perform a task.

A definition does not merely add qualities.
It tries to establish what makes the thing the thing.

4.2 Why definition matters

Definition matters because expression often needs more than depiction.

We need to know:

  • what a key term means in the current discussion

  • which features are essential and which are secondary

  • where the boundary of the term lies

  • how definition differs from mere description

Without definition, language can remain too loose. A person may speak about system, freedom, health, tool, problem, or success without making clear what exactly is being included. Description may create a general impression, but definition provides a more stable conceptual frame.

4.3 Key distinctions

4.3.1 1. Defining terms

A definition often begins by naming the thing and stating what kind of thing it is.

Term Definition Comment
tool an object used to perform a task defines by function
parent a person who raises or cares for a child defines by role
room an enclosed space within a building defines by structure
risk the possibility of harm or unwanted outcome defines by condition

Examples:

  • A chair is a piece of furniture used for sitting.

  • A plant is a living thing that grows in the natural world.

  • A rule is a limit or expectation that guides action.

  • Cooperation is working with others toward a shared result.

This matters because a term often becomes usable only when its role or type is stated clearly.

4.3.2 2. Essential features

A good definition identifies features that are central rather than accidental.

Type of feature Description Example
essential feature a feature needed for the term to hold a triangle has three sides
non-essential feature a feature that may be common but not required a chair may be wooden, but it need not be

Examples:

  • A knife is a tool with a cutting edge.
    The cutting function is essential.

  • A chair is used for sitting.
    Being blue or metal is not essential.

  • A teacher teaches.
    Being tall, calm, or strict is not essential.

This matters because definition should focus on what makes the thing what it is, not on every trait it may happen to have.

4.3.3 3. Boundary-setting

A definition helps separate what belongs from what does not.

Function Description Example
inside the boundary clearly included by the term a hammer as a tool
outside the boundary clearly excluded a tree as not a tool
borderline case a case requiring further clarification is software a tool? depends on context and definition

Examples:

  • If a tool is defined as something used to perform a task, a hammer belongs inside the boundary.

  • If a parent is defined by care and raising, mere biological relation may not always be enough in every context.

  • If a room is defined as an enclosed space, an open balcony may fall outside.

This matters because definition is not only about stating features. It is also about drawing a line.

4.3.4 4. Definition vs. loose description

Description and definition may look similar, but they do different work.

Loose description Definition
A chair is usually small, useful, and found in rooms. A chair is a piece of furniture designed for sitting.
A teacher often stands in front of a class and explains things. A teacher is a person whose role is to teach.
A tool is often made of metal or wood and helps with work. A tool is an object used to perform a task.

Examples:

  • The room is quiet and bright describes it.

  • A room is an enclosed space within a building defines it.

  • He is calm and speaks clearly describes him.

  • A teacher is a person whose role is to teach defines the role.

This matters because description shows how something appears, while definition states what it is in a more bounded way.

4.4 Core patterns

This chapter rests on a few basic patterns.

4.4.1 Basic expressive patterns

  • Definition states what something is.

  • A definition often identifies a kind and a distinguishing feature.

  • Essential features matter more than accidental traits.

  • Definition sets a boundary.

  • Definition is stricter than loose description.

4.4.2 Basic language patterns

  • A ___ is a ___ that ___.

  • ___ means ___.

  • By ___, we mean ___.

  • The essential feature of ___ is ___.

  • This definition includes ___ but excludes ___.

  • This is a description, not yet a definition.

4.5 Definition and the movement of this volume

Definition strengthens the work begun by statement and description.

4.5.1 Statement establishes the claim

  • This is a tool.

4.5.2 Description makes it perceptible

  • It is small, metal, and used by hand.

4.5.3 Definition stabilizes the term

  • A tool is an object used to perform a task.

This is a major structural step. Expression is no longer only saying and showing. It is now setting conceptual boundaries.

4.6 Examples

Here are some clean examples of definition in use.

4.6.1 Direct definitions

  • A plant is a living thing that grows in the natural world.

  • A parent is a person who cares for and raises a child.

  • A rule is a limit or expectation that guides action.

  • A risk is the possibility of harm or unwanted outcome.

4.6.2 Definitions with essential feature

  • A knife is a tool designed for cutting.

  • A room is an enclosed space within a building.

  • A process is an ordered series of steps or changes.

  • Cooperation is working together toward a shared result.

4.6.3 Boundary-setting examples

  • A chair is furniture for sitting, not just any object in a room.

  • A teacher is defined by role, not merely by age or knowledge alone.

  • A tool is defined by use, not by material alone.

  • A family member may be defined differently depending on whether the focus is biological, legal, or relational.

These examples show that definition often includes:

  • term-fixing

  • essential feature

  • conceptual boundary

  • inclusion and exclusion

  • stronger stability than description alone

4.7 Why this matters

To understand definition is to understand how expression becomes conceptually stable.

Without definition, language may still describe, but it cannot always hold its own terms firmly. A reader may sense what is meant without knowing exactly what is included, excluded, or treated as central. Definition gives expression stronger structure by reducing that uncertainty.

Definition is also essential because many later structures depend on it:

  • classification depends on stable definitions

  • comparison becomes stronger when the compared terms are clearly bounded

  • explanation improves when its key terms are defined

  • evaluation becomes more rigorous when the object of judgment is fixed

  • a whole thought becomes more coherent when its central terms are stable

Structured expression therefore requires more than vivid description. It also requires the discipline of saying what a thing is.

4.8 Chapter summary

This chapter introduces definition as the expressive act that directly states what something is.

Unlike loose description, definition identifies essential features and sets a boundary. It helps stabilize terms, clarify concepts, and distinguish what belongs from what does not.

The central lessons of this chapter are:

  • some forms of expression must define, not merely describe

  • definition states what something is

  • essential features matter more than accidental traits

  • definition helps set boundaries

  • definition differs from loose description

  • conceptual stability depends on clear definition

4.8.1 Core Knowledge (Must Remember)

Core Idea Key Insight Must Remember
Definition States what something is, not just how it appears Sets a conceptual boundary; crucial for stable terms
Essential vs. accidental Essential features make the thing what it is A chair is for sitting; being wooden is accidental
Boundary-setting Separates what belongs from what does not Definition includes some cases and excludes others
Definition vs. description Description shows; definition delimits A term is not truly fixed until it is defined, not merely depicted
“A ___ is a ___ that ___” The basic definition pattern Kind + distinguishing feature = strong definition

To recognize definition is to strengthen expression at one of its most important points: the moment where language stops merely showing and begins to delimit.
